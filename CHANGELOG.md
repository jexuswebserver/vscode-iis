# Change Log

All notable changes to the "IIS and IIS Express extension for Visual Studio Code" extension are documented in this file.

The format is intentionally simple and VS Code Marketplace friendly: newest release first, with short notes for each published version.

## 1.0.15 - 2026-04-01

- Restricted releases to Windows only.
- Updated dependencies.

## 1.0.14 - 2024-01-05

- Removed the pop-up.

## 1.0.13 - 2024-01-05

- Fixed default file resolution.

## 1.0.12 - 2024-01-05

- Stopped overwriting the configured path.
- Revised text to avoid confusion.

## 1.0.11 - 2024-01-05

- Fixed a few bugs.
- Added a documentation link.

## 1.0.10 - 2023-04-09

- Moved rating prompt to the end of initialization.

## 1.0.9 - 2023-04-09

- Added rating collection.

## 1.0.8 - 2023-03-30

- Added a second status bar item when a workspace is opened.

## 1.0.7 - 2023-03-29

- Fixed bugs in workspace support.

## 1.0.6 - 2023-03-29

- Added full workspace support so that multiple folders can be used.

## 1.0.5 - 2023-03-29

- Added new setting `iis.browser`.

## 1.0.4 - 2023-03-29

- Revised file selection.
- Revised working directory of Jexus Manager.

## 1.0.3 - 2023-03-29

- Stopped temp config file from showing if other config files exist.

## 1.0.2 - 2023-03-24

- Removed commands from Command Pallette on non-supported OS.

## 1.0.1 - 2023-03-24

- Revised UI elements.
- Changed default config file path.
- Added OS detection.

## 1.0.0 - 2023-03-23

Initial release.

### Changes

- Added `.config` file detection in current workspace.
- Added initial integration with Jexus Manager in IIS Express mode.

### Known Issues

- No IIS mode yet.
- ASP.NET Core projects might not work due to Jexus Manager limitations.
